## 👨‍💻 عن المطور

**Eng. LoL ✨** - مطور النظام

نظام **LOOK CASHIER** هو منصة متكاملة لإدارة المراكز التعليمية والمعاهد، تم تطويره بعناية فائقة لتوفير حل شامل وسهل الاستخدام لجميع احتياجات إدارة المراكز التعليمية.

## 🎯 نبذة عن المشروع

**LOOK CASHIER** هو نظام متطور لإدارة المراكز التعليمية يوفر:

- 📊 **إدارة مالية متكاملة** - تتبع الإيرادات والمصروفات
- 👥 **إدارة الطلاب والمستخدمين** - نظام صلاحيات متقدم
- 📝 **إدارة الفواتير** - فواتير للطلاب المشتركين والفرديين
- 💳 **إدارة الاشتراكات** - أسعار مرنة وقابلة للتخصيص
- 📈 **تقارير شاملة** - تحليلات معمقة للأداء المالي
- ⏰ **تتبع الطلاب المتأخرين** - نظام تنبيهات ذكي
- 📅 **تقفيل يومي** - إغلاق الحسابات بشكل آمن

## 📥 التثبيت والتشغيل

### الطريقة الأولى: التثبيت من ملف EXE
1. قم بتحميل ملف `LOOK-CASHIER-Setup.exe` من الإصدارات
2. قم بتشغيل الملف
3. اتبع تعليمات المثبت
4. قم بتشغيل البرنامج من قائمة البداية

### الطريقة الثانية: من المصدر
```bash
# 1. استنساخ المستودع
git clone https://github.com/yourusername/look-cashier.git

# 2. الانتقال للمجلد
cd look-cashier

# 3. تثبيت المكتبات
npm install

# 4. تشغيل البرنامج
npm start
```

## 🔧 البناء والتجميع

### بناء EXE للـ Windows
```bash
# بناء لكل إصدارات Windows
npm run build

# أو لإصدار محدد
npm run build:win7
npm run build:win10
npm run build:win11
```

### الملفات الناتجة
- `LOOK-CASHIER-Setup-x64.exe` - للـ 64-bit
- `LOOK-CASHIER-Setup-x86.exe` - للـ 32-bit
- `LOOK-CASHIER-Portable.exe` - نسخة محمولة
