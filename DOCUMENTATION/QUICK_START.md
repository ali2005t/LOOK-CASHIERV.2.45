# ⚡ البدء السريع (Quick Start)

## 🚀 بناء التطبيق في 3 خطوات

### الخطوة 1: تثبيت المكتبات
```bash
npm install
```

### الخطوة 2: تشغيل البرنامج
```bash
npm start
```

### الخطوة 3: بناء الملفات
```bash
npm run build
```

---

## 🖱️ طريقة سهلة (Windows)

### انقر على ملف البناء
1. افتح مجلد المشروع
2. ابحث عن `build.bat`
3. انقر عليه بنقرة مزدوجة
4. اختر نوع البناء

---

## 📦 الملفات الناتجة

بعد البناء، ستجد الملفات في مجلد `install/`:

```
LOOK-CASHIER-Setup-2.0.0-x64.exe      (64-bit)
LOOK-CASHIER-Setup-2.0.0-ia32.exe     (32-bit)
LOOK-CASHIER-Setup-2.0.0-arm64.exe    (ARM)
```
