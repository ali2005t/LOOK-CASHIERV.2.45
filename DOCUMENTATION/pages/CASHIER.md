# 💵 الكاشير (Cashier)

## نبذة عامة
صفحة الكاشير هي المكان الرئيسي لإنشاء الفواتير الجديدة للطلاب، سواء كانوا مشتركين أو يريدون ملازم فردية.

## 🎯 الوظائف الأساسية

### 1. البحث عن الطالب
- ابحث برقم الطالب أو الاسم
- اختر الطالب من القائمة
- سيتم عرض معلومات الطالب تلقائياً

### 2. إضافة العناصر
- **الاشتراكات**: اختر من قائمة الاشتراكات المتاحة
- **ملازم فردية**: أضف كمية وسعر للملازم
- **خدمات إضافية**: إضافة رسوم إضافية أخرى

### 3. حساب المبلغ
- السعر الإجمالي يُحسب تلقائياً
- يمكنك تطبيق خصومات يدوية
- عرض الفرق في السداد إن وجد

## 📋 خطوات الاستخدام

### خطوة 1: البحث عن الطالب
```
1. انقر على حقل "البحث عن الطالب"
2. اكتب اسم أو رقم الطالب
3. اختر الطالب من النتائج
```

### خطوة 2: اختيار الخدمة
```
1. حدد نوع الخدمة (اشتراك أو ملزمة)
2. اختر من القائمة
3. سيظهر السعر تلقائياً
```

### خطوة 3: إنهاء الفاتورة
```
1. تحقق من المبلغ
2. اختر طريقة الدفع
3. اضغط "إنشاء فاتورة"
4. اطبع الفاتورة أو احفظها
```

## 💳 طرق الدفع المدعومة

- 💵 **نقداً** (الدفع على المكان)
- 🏦 **تحويل بنكي** (رقم الفاتورة كمرجع)
- 📱 **محافظ إلكترونية** (Apple Pay, Google Pay)
- 🎫 **شيك** (لاحقاً)

## 🔧 الخيارات المتقدمة

### الخصومات
- خصم نسبة مئوية
- خصم مبلغ ثابت
- خصم عام للفاتورة

### الملاحظات
- أضف ملاحظات للفاتورة
- سيتم حفظها مع الفاتورة

### إرسال الفاتورة
- اطبع الفاتورة مباشرة
- أرسل بالبريد الإلكتروني
- احفظ كـ PDF

## ⚡ اختصارات مفيدة

| المفتاح | الوظيفة |
|--------|--------|
| `Ctrl+N` | فاتورة جديدة |
| `Ctrl+P` | طباعة |
| `Ctrl+S` | حفظ |
| `F2` | البحث السريع |

## ⚠️ التنبيهات المهمة

### تنبيهات الأخطاء
- ⚠️ إذا كان الطالب مديون
- ⚠️ إذا انتهت الاشتراكية
- ⚠️ إذا فقد الطالب صلاحية الشراء

## 📊 إحصائيات الكاشير

- عدد الفواتير اليومية
- إجمالي المبيعات
- المبالغ المدفوعة والمعلقة

## ❓ أسئلة شائعة

**س: ماذا لو نسيت البيانات؟**
ج: يمكنك العودة لآخر فاتورة من سجل الفواتير

**س: هل يمكن تعديل الفاتورة؟**
ج: نعم، من صفحة الفواتير السابقة

**س: كيف أطبع الفاتورة؟**
ج: انقر على زر الطباعة بعد الانتهاء
