# 👥 إدارة المستخدمين (Users)

## الوصف
إدارة حسابات المستخدمين والصلاحيات والأدوار المختلفة.

## 👤 أنواع المستخدمين

### 1. المسؤول (Admin)
- صلاحيات كاملة
- إدارة جميع الحسابات
- تعديل الإعدادات العامة

### 2. محاسب (Accountant)
- إدارة الفواتير والمبيعات
- عرض التقارير المالية
- لا يمكن حذف البيانات

### 3. كاشير (Cashier)
- إنشاء فواتير فقط
- لا يمكن عرض المراكز الأخرى
- دخول محدود

### 4. موظف إداري (Admin Staff)
- إدارة الطلاب
- لا يمكن الوصول للبيانات المالية

## 🔐 الصلاحيات

### صلاحيات العمليات
- عرض الفواتير
- إنشاء فواتير
- تعديل الفواتير
- حذف الفواتير
- عرض التقارير
- إدارة المستخدمين
- إدارة الإعدادات

## ➕ إضافة مستخدم جديد

### الخطوات
1. انقر "إضافة مستخدم جديد"
2. أدخل البيانات الأساسية
3. اختر الدور
4. حدد الصلاحيات
5. احفظ التغييرات

### البيانات المطلوبة
- الاسم الكامل
- البريد الإلكتروني
- كلمة المرور الأولية
- نوع الدور

## 🔑 إدارة كلمات المرور

- تغيير كلمة المرور
- إعادة تعيين كلمة المرور
- إجبار تغيير كلمة المرور عند الدخول الأول

## ⚠️ الأمان

- تفعيل المصادقة الثنائية
- حظر الحسابات غير النشطة
- تسجيل جميع العمليات
- إعدادات تنبيه الأمان
