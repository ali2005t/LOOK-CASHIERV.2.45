# 🧾 الطلاب المشتركين (Invoices)

## الوصف
صفحة الطلاب المشتركين توفر قائمة شاملة بجميع فواتير الطلاب المشتركين، مع إمكانيات بحث وتصفية متقدمة.

## 🔍 البحث والتصفية

### أنواع البحث
- البحث برقم الطالب
- البحث باسم الطالب
- البحث برقم الفاتورة
- البحث بنوع الاشتراك

### التصفية
- حسب الفرقة الدراسية
- حسب نوع الاشتراك
- حسب حالة الدفع
- حسب التاريخ

## 📋 العمليات الممكنة

### 1. عرض التفاصيل
- اضغط على أي فاتورة
- عرض جميع البيانات
- تاريخ الفاتورة والمبلغ

### 2. طباعة الفاتورة
- اختر الفاتورة
- اضغط "طباعة"
- اختر الطابعة

### 3. تحميل PDF
- احفظ الفاتورة كـ PDF
- لإرسالها للطالب

### 4. تعديل الفاتورة
- يمكن تصحيح الأخطاء
- لا يمكن تغيير المبلغ إذا تم الدفع

### 5. حذف الفاتورة
- حذف آمن مع التأكيد
- سيتم حفظ نسخة احتياطية

## 📊 الإحصائيات

- إجمالي الفواتير
- المبالغ المسددة
- المبالغ المعلقة
- متوسط الفاتورة

## 🎯 الاستخدام اليومي

1. افتح الصفحة
2. ابحث عن الطالب
3. عرض الفواتير السابقة
4. تحقق من حالة الدفع
5. اطبع أو عدّل إن لزم

## 📱 على الهاتف

- واجهة محسّنة للهاتف
- يمكن الطباعة عبر AirPrint
- تحميل سريع للملفات
