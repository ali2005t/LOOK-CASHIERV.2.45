# 💳 الاشتراكات والأسعار (Subscriptions)

## الوصف
إدارة جميع أنواع الاشتراكات والأسعار المختلفة للكورسات والملازم.

## 📝 الاشتراكات الأساسية

### أنواع الاشتراكات
- اشتراكات كاملة (جميع الفرق)
- اشتراكات فرقة واحدة
- اشتراكات موسمية
- اشتراكات خاصة

## 💰 إدارة الأسعار

### تعديل الأسعار
1. اختر الاشتراك
2. حدد السعر الجديد
3. اختر تاريخ البدء
4. احفظ التغييرات

### الأسعار المختلفة
- سعر موحد
- أسعار حسب الفرقة
- خصومات جماعية
- أسعار مواسمية

## 🎛️ التحكم

### تفعيل/تعطيل
- إظهار الاشتراك
- إخفاء الاشتراك (غير متاح)
- حذف نهائي

### الترتيب
- تغيير ترتيب الاشتراكات
- تحديد الأولويات

## 📊 الإحصائيات

- عدد المشتركين لكل اشتراك
- الإيرادات من كل اشتراك
- أكثر الاشتراكات شهرة
