دليل المساعدة الشامل

👋 أهلاً وسهلاً بك!
ستجد هنا شرحًا مفصلاً وسهلاً لكل صفحة من صفحات النظام، مع معلومات عملية عن كيفية استخدام كل ميزة. النظام مصمم ليكون سهل الاستخدام، لكن إذا احتجت مساعدة، كل شيء مشروح هنا بوضوح.

الصفحات الرئيسية

لوحة التحكم

  • 📊 تعرض ملخص سريع لأهم الإحصائيات (عدد الطلاب، الإيرادات، أحدث الفواتير).
  • 👁️ رؤية شاملة وسريعة لحالة السنتر الحالية.
  • ⚡ تحديث فوري للبيانات الأساسية.

الكاشير

  • 💵 إنشاء فواتير جديدة للطلاب المشتركين والغير مشتركين.
  • 📝 إضافة اشتراكات أو ملازم فردية بسهولة.
  • 🔍 البحث السريع عن طالب وإضافة فاتورة له مباشرة.

إدارة الفواتير والطلاب

الطلاب المشتركين

  • 📋 استعراض وإدارة جميع فواتير الطلاب المشتركين.
  • 🔎 البحث والتصفية المتقدمة حسب الاسم أو الفرقة أو نوع الاشتراك.
  • 🖨️ فتح تفاصيل الفاتورة وطباعتها مباشرة.
  • ✏️ تعديل وحذف الفواتير بسهولة.

فواتير فردية

  • 👤 إدارة ملازم أو محاضرات فردية للطلاب غير المشتركين.
  • 🔍 البحث والتصفية لسهولة إيجاد الفواتير.
  • 💾 حفظ سجل كامل للفواتير الفردية.

الإعدادات والاشتراكات

الاشتراكات والأسعار

  • 💳 ضبط أسعار الكورسات والملازم المختلفة.
  • ➕ إضافة أو تعديل اشتراكات جديدة.
  • 📊 تحديد الأسعار لكل فرقة دراسية.
  • 📱 إدارة الاشتراكات ونشرها وإخفاؤها.

الإعدادات

  • 🏢 ضبط بيانات السنتر (الاسم، العنوان، الشعار).
  • 📱 تحديث أرقام الهاتف والبريد الإلكتروني.
  • ⚙️ إعدادات النظام العامة والخيارات المتقدمة.

التقارير والمستخدمين

التقارير

  • 📈 استخراج تقارير الأرباح والإيرادات الشاملة.
  • 📊 توزيع الطلاب حسب الاشتراكات والفرقات.
  • 💹 تحليل متقدم للإيرادات والمصروفات.
  • 📥 تحميل التقارير بصيغ مختلفة.

إدارة المستخدمين

  • 👥 إضافة وتعديل وحذف المستخدمين.
  • 🔐 التحكم الدقيق في الصلاحيات والأدوار.
  • ✅ تفعيل أو تعطيل الحسابات.
  • 🛡️ إدارة أذونات العمليات المختلفة.

العمليات اليومية

المصروفات

  • 💸 تسجيل المصروفات اليومية بكل سهولة.
  • 📝 استعراض وتنظيم ملخص المصروفات الشامل.
  • 🔍 البحث والتصفية المتقدمة للمصروفات.
  • 📊 عرض إحصائيات المصروفات الشهرية.

الطلاب المتأخرين في الدفع

  • ⚠️ عرض قائمة الطلاب الذين تأخروا عن السداد.
  • 🎨 ملون حسب درجة التأخير (أحمر/أصفر/أخضر).
  • 💰 معلومات عن المبلغ المتبقي والفترة المتبقية.
  • 🔔 تنبيهات سريعة للمتأخرين.

التقفيل اليومي

  • ✅ تقفيل الحسابات اليومية بشكل آمن وموثوق.
  • 📊 عرض ملخص شامل للإيرادات اليومية.
  • 🖨️ طباعة التقرير النهائي للمراجعة والأرشفة.
  • 📱 ربط مع فاتورة فودافون للتسديدات الإلكترونية.
